# 🛰️ Demand Intelligence System

An end-to-end Machine Learning pipeline and futuristic Streamlit web dashboard for predicting retail sales and detecting supply chain anomalies.

## 🚀 Features
* **Sales Overview:** Interactive Plotly charts tracking 4 years of historical data.
* **XGBoost Forecasting:** Machine learning engine predicting 3-month future trajectories.
* **Anomaly Detection:** Isolation Forest algorithm catching unexpected supply chain shocks.
* **K-Means Clustering:** Automated product segmentation for optimized warehouse stocking.

## 🛠️ Tech Stack
* **Python** (Pandas, NumPy)
* **Machine Learning** (Scikit-Learn, XGBoost)
* **Frontend** (Streamlit, Plotly, Custom CSS)
