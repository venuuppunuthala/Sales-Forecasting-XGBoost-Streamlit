# 🛰️ Demand Intelligence System

An end-to-end Machine Learning pipeline and futuristic Streamlit web dashboard for predicting retail sales and detecting supply chain anomalies using XGBoost, Isolation Forest, and K-Means Clustering.

## 🚀 Key Features

* **Sales Overview Dashboard:** Interactive Plotly visual analytics tracking 4 years of historical sales trends, yearly revenue, and regional category performance.
* **XGBoost Forecast Explorer:** High-precision predictive engine generating 3-month forward projections for different product categories and regions.
* **Anomaly Detection Engine:** Automated threat and anomaly reporting using Isolation Forest and Z-Score analysis to catch volume spikes and demand shocks.
* **K-Means Product Segmentation:** Automated cluster analysis grouping products by volume, volatility, order value, and YoY growth parameters.
* **Futuristic UI/UX:** Dark-mode cyber aesthetic enhanced with custom HTML/CSS and interactive Plotly visualization modules.

## 🛠️ Tech Stack

* **Language:** Python
* **Data Processing:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn, XGBoost, Isolation Forest, K-Means Clustering
* **Visualization:** Plotly, Matplotlib, Seaborn
* **Web Framework & Styling:** Streamlit, Custom CSS

## 📂 Repository Contents

* `app.py`: Main Streamlit application file containing the multi-page dashboard layout and custom UI code.
* `analysis.ipynb`: Comprehensive Jupyter Notebook containing initial exploratory data analysis, model development, and validation experiments.
* `train.csv`: Primary historical sales dataset used for training models and generating dashboard visualizations.
* `requirements.txt`: List of dependencies and Python packages required to deploy and execute the web application.
* `Summary.pdf`: Executive business report summarizing project insights, forecasting performance, and strategic deployment recommendations.
* `charts/` (and image assets): Exported visual plots and dashboard UI screenshots showcasing the system capabilities.
